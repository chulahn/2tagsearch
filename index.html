2 Tag Search

Find Instagram posts that have both the hashtags you are looking for.

How does it work?


  1. Finds all the images that have the first hashtag.
  2. From those results, filter the images that contain the second hashtag.
  3. Repeat except search images with second hashtag, and filter the result images that have the first hashtag.
